Valorant

Tips and Tricks!

Valorant is a competitive 5v5 FPS on PC only, its just like CS:GO with the only difference really being that there is different agents with different abilities and there is different agent types such as Controller, Sentinel, Initiator and Duelist. Each different agent type has its own playstyle.
